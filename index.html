🚗 Guia Completo — AutoGestor Premium

Vou dividir em 4 partes. Faça uma parte de cada vez, na ordem.

PARTE 1 — Instalar os programas necessários

Você precisa instalar 2 programas no seu computador. Só precisa fazer isso uma vez.

Passo 1 — Instalar o Node.js

  1. Abra o navegador e acesse: https://nodejs.org
  2. Clique no botão verde grande que diz "LTS" (é a versão recomendada)
  3. Vai baixar um arquivo .msi (Windows) ou .pkg (Mac)
  4. Abra o arquivo baixado e clique em "Next" em todas as telas, depois "Install"
  5. Quando terminar, clique em "Finish"

Como confirmar que funcionou:

  • Aperte Windows + R, digite cmd, aperte Enter
  • Na janela preta que abriu, digite exatamente: node --version e aperte Enter
  • Tem que aparecer um número como v20.11.0 (qualquer número está ok)

Passo 2 — Instalar o VS Code (editor de código)

  1. Acesse: https://code.visualstudio.com
  2. Clique em "Download for Windows" (ou Mac/Linux)
  3. Abra o arquivo baixado e instale normalmente clicando em "Next" até o fim

PARTE 2 — Criar o projeto Firebase (banco de dados gratuito)

O Firebase é onde os dados do seu sistema ficam guardados. É gratuito.

Passo 3 — Criar conta no Firebase

  1. Acesse: https://console.firebase.google.com
  2. Faça login com sua conta Google (a mesma que você usa no Gmail)
  3. Clique em "Criar um projeto"

Passo 4 — Configurar o projeto Firebase

  1. Nome do projeto: Digite autogestor (ou qualquer nome que quiser)
  2. Clique em "Continuar"
  3. Na tela do Google Analytics, pode deixar desativado e clicar em "Criar projeto"
  4. Aguarde alguns segundos até aparecer "Seu novo projeto está pronto"
  5. Clique em "Continuar"

Passo 5 — Ativar o Login com Google

  1. No menu da esquerda, clique em "Authentication"
  2. Clique em "Vamos começar"
  3. Clique em "Google" (na lista de provedores)
  4. Clique no botão para ativar (o botão fica azul)
  5. No campo "E-mail de suporte do projeto", selecione seu e-mail
  6. Clique em "Salvar"

Passo 6 — Criar o banco de dados (Firestore)

  1. No menu da esquerda, clique em "Firestore Database"
  2. Clique em "Criar banco de dados"
  3. Selecione "Iniciar no modo de produção" e clique em "Avançar"
  4. Escolha a localização: "southamerica-east1" (São Paulo — mais rápido para o Brasil)
  5. Clique em "Ativar" e aguarde

Passo 7 — Criar o armazenamento de arquivos (Storage)

  1. No menu da esquerda, clique em "Storage"
  2. Clique em "Vamos começar"
  3. Clique em "Avançar" e depois em "Concluído"

Passo 8 — Pegar as credenciais do Firebase

Estas são as "chaves" que ligam o sistema ao seu banco de dados. Guarde com cuidado.

  1. Clique na engrenagem ⚙️ no menu da esquerda (ao lado de "Visão geral do projeto")
  2. Clique em "Configurações do projeto"
  3. Role a página para baixo até ver "Seus apps"
  4. Clique no ícone </> (Web)
  5. Em "Apelido do app", digite autogestor-web
  6. Não marque o Firebase Hosting
  7. Clique em "Registrar app"
  8. Vai aparecer um bloco de código assim:
js
const firebaseConfig = {
  apiKey: "AIzaSy...",
  authDomain: "autogestor.firebaseapp.com",
  projectId: "autogestor",
  storageBucket: "autogestor.appspot.com",
  messagingSenderId: "123456789",
  appId: "1:123456789:web:abc123"
}
  1. Copie esse bloco inteiro (selecione tudo e Ctrl+C)
  2. Clique em "Continuar no console"

PARTE 3 — Configurar e rodar o sistema

Passo 9 — Extrair o ZIP baixado

  1. Localize o arquivo autogestor-premium.zip que você baixou
  2. Clique com o botão direito nele → "Extrair tudo"
  3. Escolha uma pasta fácil, como C:\projetos\autogestor
  4. Clique em "Extrair"

Passo 10 — Abrir o projeto no VS Code

  1. Abra o VS Code
  2. Clique em "File" → "Open Folder"
  3. Navegue até a pasta autogestor que você extraiu e clique em "Selecionar pasta"

Passo 11 — Colar as credenciais do Firebase

  1. No VS Code, no painel da esquerda, clique no arquivo src/firebase.js
  2. Localize este trecho (linha 20 aproximadamente):
js
const firebaseConfig = {
  apiKey: "COLE_SUA_API_KEY_AQUI",
  authDomain: "SEU_PROJETO.firebaseapp.com",
  ...
  1. Selecione apenas esse bloco (do const firebaseConfig = { até o } que fecha)
  2. Cole o bloco que você copiou do Firebase (Ctrl+V)
  3. Salve o arquivo: Ctrl+S

Passo 12 — Abrir o terminal no VS Code

  1. No VS Code, clique em "Terminal" no menu superior
  2. Clique em "New Terminal"
  3. Uma janela preta vai aparecer na parte de baixo

Passo 13 — Instalar as dependências

No terminal, digite o comando abaixo e aperte Enter:

npm install

Vai aparecer muito texto rolando. Aguarde até parar. Pode demorar de 1 a 3 minutos. Quando terminar, o cursor vai piscar novamente.

Passo 14 — Rodar o sistema

No terminal, digite e aperte Enter:

npm run dev

Vai aparecer algo como:

  ➜  Local:   http://localhost:5173/

Agora abra seu navegador e acesse: http://localhost:5173

🎉 O sistema vai aparecer! Faça login com sua conta Google.

PARTE 4 — Aplicar as regras de segurança do banco

Isso garante que só você acessa seus dados.

Passo 15 — Regras do Firestore

  1. Abra o Firebase Console: https://console.firebase.google.com
  2. Clique em "Firestore Database" → aba "Regras"
  3. Apague todo o conteúdo que está lá
  4. Abra o arquivo firestore.rules do seu projeto no VS Code
  5. Copie todo o conteúdo (Ctrl+A depois Ctrl+C)
  6. Cole no Firebase (Ctrl+V)
  7. Clique em "Publicar"

Passo 16 — Regras do Storage

  1. No Firebase Console, clique em "Storage" → aba "Regras"
  2. Apague tudo que está lá
  3. Abra o arquivo storage.rules no VS Code, copie tudo e cole
  4. Clique em "Publicar"
